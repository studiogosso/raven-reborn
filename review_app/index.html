モブ出現ルール

  • 初期出現数: 8（通常）/ 5（スプリント）
  • 最大同時存在: 20
  • 毎ターン+1体出現
  • ターン1-7: ランクE-D（インデックス0-3）
  • ターン8-19: ランクE-C（インデックス1-5）
  • ターン20+: ランクD-B（インデックス3-7）

モブ行動

  • 毎ターン40%の確率で行動スキップ
  • 最寄りのプレイヤー/ボットに向かって移動（A*探索）
  • ノイズヒートマップに引き寄せられる（閾値0.3）
  • 囮に引き寄せられる（より近い場合は優先）
  • 50%ダメージルール: モブは人間プレイヤーにATKの半分、ボットに全ATK
  • モブドロップ率: E=20%, D=30%, C=40%, B=50%
  • ドロップ報酬: E: XP+1 / D: XP+2またはHP+1 / C: XP+3またはT1カード / B: XP+5またはT2カード

ミミック

  • HP:3 ATK:2 XP:3 ランク:C
  • 移動せず、隣接ターゲットのみ攻撃
  • 人間に50%ダメージ、ボットにフルダメージ
  • 撃破時にティア3以上のカードをドロップ
  • 宝箱イベント時に出現（イベントあたりミミック1体）

ボス: 深淵の守護者

  • HP:12 ATK:4
  • イベントシステムで出現（ランダム、ゲーム開始時ではない）
  • 出現位置: マップ中央（ボス部屋）
  • 毎ターン隣接する全プレイヤー/ボットを攻撃
  • 撃破時に旗をドロップ、XP+15 と VP+5を獲得（bossKill経由）
ボスフェーズシステム（v1.8）
  • フェーズ1（HP < 75%）: 移動速度+1（2歩/ターン）
  • フェーズ2（HP < 50%）: 範囲2以内に全体1ダメージ
  • フェーズ3（HP < 25%）: 毎ターンノイズ5を発生（全体警報）

同一ターン内に特定のカードタイプの組み合わせを使用すると、ターン終了時にシナジーボーナスが発動:

コンボシステム

  • コンボ数 = カード使用数 + キル数 + 部屋発見数 + 旗獲得数（1ターンあたり）
  • 3行動以上: COMBO! XP 1.5倍
  • 4行動以上: ULTRA COMBO! XP 2.0倍

その他のコンボ

  • 反響+雷: 反響デバフ付きの敵に雷ダメージ1.5倍
  • 水+雷: 水タイル上で3倍ダメージ
  • 草+火: 草への火/雷で燃焼タイル生成
  • ステルス緩和: カード0枚使用 + 1歩以下で完全ステルス（ターンノイズリセット）

ゲームシステム▼

VP / 勝利条件

  • 通常モード: 30 VPで勝利（40x40マップ）
  • スプリントモード: 15 VPで勝利（25x25マップ）
  • VP獲得源:
    • VPカード: 1/3/6 VP（屋敷/公領/属州）
    • 庭園: デッキ5枚ごとに1 VP
    • 旗配達: 10 VP（3ターン防衛後）
    • プレイヤーキル: 3 VP（各）
    • ボス撃破: 5 VP
    • スカウトVP: 部屋発見ごとに1 VP
    • 探索ボーナス: 全部屋発見で3 VP
    • 英雄の証カード: 即座に5 VP
    • 呪いカード: -1 VP（各）

ターン構造

  • 開始時: AP=1、購入=1（+クラスボーナス）、MP=3（+レベル+クラスボーナス）
  • XP減衰: 毎ターン-1（最小0）
  • 手札に5枚ドロー
  • フリーアクション: 移動（MP）、カード使用（AP）、サプライから購入（購入数+XP）
  • ターン終了: 手札を捨て、コンボ/シナジー判定後、ボット/敵のターン
  • 自動終了: AP=0、MP=0、使用可能カードなしで発動（300msディレイ）

ノイズシステム

  • 各歩: ノイズ+1（水: +2、草: 次の歩ノイズ0）
  • カードノイズ: カードごとに異なる
  • ノイズヒートマップ: ミニマップ上に表示（切替可能）
  • モブはノイズの高い地域に引き寄せられる（閾値0.3）
  • ノイズ閾値: 3以上 = 「モブが接近中」、5以上 = 「位置が暴露」
  • 反響の間: ノイズ2倍、沈黙の間: ノイズ=0
  • 残響の罠デバフ: Nターンの間行動ごとにノイズ+2
  • 旗所持者: 常に敵に見える（大ノイズ）

戦場の霧

  • 3段階: 可視（明るい）/ 探索済み（薄暗い）/ 未探索（真っ暗）
  • 基本視野: 4タイル（視線チェック付き）
  • 見張り台: 4タイル半径の永続視野
  • 偵察カード: 一時的な範囲視野（2ターン）
  • 全知の目: 全マップ可視化（Nターン）

デッキ構築

  • 初期デッキ: 10枚（銅貨x3、小刀x2、俊足の薬x1、集中x1、見張り台x1、隠密x1、薬草x1）
  • 手札上限: 毎ターン5枚
  • 購入: XP + 購入権でサプライからカードを獲得
  • 除外: 礼拝堂カードで手札から最大2枚を永久除外
  • 改築: 手札1枚を除外し、XP+2以下のカードを獲得
  • 祭壇: カード1枚を無料除外
  • デッキシャッフル: ドロー山札が空の時、捨て札をシャッフルしてデッキに

合成システム

  • 消費: 1 AP
  • 武器 + 杖: 合計パワー、杖の射程、貫通射撃
  • 武器 + 武器: 合計パワー、最大射程、合計ノイズ
  • 杖 + 杖: 合成不可
  • 合成カードは合計AP消費とノイズを持つ
  • 合成の巻物が必要（ティア4、XP 9）

レベルアップシステム

  • コスト: (現在レベル + 1) x 2 XP
  • 毎ターン最大1回
  • 成長の書カード: 無料レベルアップ
  • 成長項目:
    • ATK: 全武器/近接に+1基本ダメージ
    • HP: 最大HP+2 かつ 即時HP+2回復
    • MP: 毎ターン基本MP+1

旗システム

  • 旗は内側リングの部屋に出現（両スタート地点から離れた場所）
  • 取得: 旗タイルを踏む
  • 旗所持者: 常に敵に見え、金色の軌跡
  • 自陣に配達: 3ターン防衛開始
  • 防衛成功: VP+10
  • 防衛失敗: 自陣を離れると旗をドロップ
  • 旗所持中の死亡: 死亡地点に旗をドロップ
  • ボス撃破時に旗をドロップ

死亡とリスポーン

  • 死亡ペナルティ: XPが0にリセット、デッキ/手札/捨て札からランダムに1枚喪失
  • 喪失カードは墓石に（他プレイヤーが拾える）
  • 自陣またはチェックポイント（設置済みの場合）でリスポーン

  • ボス出現: 赤/オレンジ60個の円形パーティクルリング（収束）

アニメーション

  • 画面揺れ: 0.3秒のcanvas振動（被ダメージ時）
  • ノイズパルス: インジケーターのbox-shadowグロー
  • スコアポップアップ: 1.2秒の浮遊+拡大パルス（「KILL!」「BOSS KILL!」「LEVEL UP!」）
  • ボス登場: 2.5秒のフルスクリーンオーバーレイ、3倍から1倍にスケール+ブラー、サブテキスト0.3秒遅延
  • クリティカル閃光: 白背景の二重点滅（各0.15秒）
  • レベルアップバースト: 紫の拡張グローリング（box-shadow 1秒）
  • カード使用演出: 0.4秒のtranslateY(-100px) + scale(0.3) + フェード
  • フローティングログ: 2.5秒のフェードイン/アウト戦闘メッセージ（画面中央）
  • フラッシュオーバーレイ: 全画面カラーフラッシュ（被ダメージ赤、回復緑、旗金）
  • ダメージ数字: マップタイル上の浮遊「-N」テキスト（40フレーム）
  • ノイズリップル: ミニマップ/マップ上の拡散円
  • 攻撃フェード: 攻撃後のタイルハイライトフェード
  • 旗の軌跡: 旗所持者の後ろに金色タイル（3ターン）
  • 足跡: 10ターン分の移動軌跡表示
