## 1. 世界設定コア

### 「神は死んだ。その死が世界の生であった。」

遥かな過去、唯一の超越的存在——**神**——は自らの存在が苦痛であることを悟り、**非存在（虚無）への意志**を持った。しかし超越的存在は即座に消滅できない。神は自己を**無数の断片に分解**することで自殺を遂げた。

この分解こそが世界の始まりであり、現在の宇宙は**死にゆく神の残骸**である。

全ての存在——石も、炎も、生き物も——は神の断片であり、それぞれが**非存在へ向かう運動**を内蔵している。これが宇宙の根本法則：**力の弱体化**。

### ゾティーク——最後の大陸

その宇宙の終わりの時代。太陽は暗く曇ったランプのように輝き、かつての大陸は全て沈んだ。最後に残った一つの陸塊——**ゾティーク**。

ここでは科学は完全に忘れ去られ、代わりに**死霊術**と**暗黒魔術**が技術の役割を果たす。死者は蘇らされて奴隷として使役され、暴虐な王と邪悪な僧侶が退廃した王朝を支配する。

**「日々は黄昏のようであり、真昼でさえ荒廃の色に染められていた。」**

## 2. ゲームへの変換

### 2.1 ダンジョンの本質

ダンジョンは建造物ではない。**死んだ神の体内**である。

プレイヤーが階層を下るということは、神の遺体のより深い部分——より原初的な断片——に近づくことを意味する。深層に行くほど、力は古く、純粋で、そして危険になる。

| 階層 | 意味 | 環境テーマ |
|------|------|-----------|
| 地表 | 神の皮膚 | 崩壊した大聖堂、枯れた王座 |
| 浅層 | 神の肉 | 有機的な洞窟、脈動する壁 |
| 中層 | 神の骨 | 結晶化した構造、巨大な骨格 |
| 深層 | 神の血管 | 流れる暗黒の液体、かつての生命力の残滓 |
| 最深部 | 神の心臓 | 鼓動を止めた巨大な器官。ここに「救済」がある |

### 2.2 死の意味

マインレンダーに従えば、**死は救済**である。しかしゾティークの世界では、**死霊術によって死後すら安息がない**。

- **通常の死**: リスポーン（神の断片として再配置される）
- **死霊術による蘇生**: 意識を持ったまま奴隷化される（最悪の運命）
- **真の死（救済）**: 完全なる非存在。ゲームの究極の到達点

### 2.3 装備の哲学

全てのアイテムは**神の断片**である。

- **武器**: 神の意志の破片。使うたびに力が弱体化する（耐久度）
- **防具**: 神の皮膚の欠片。装着者を守るが、同時に存在に縛り付ける
- **呪われたアイテム**: 自我を持った神の断片（付喪神の概念と融合）。強力だが、使用者の意志を蝕む

### 2.4 敵の本質

| 敵タイプ | 世界観的意味 |
|---------|-------------|
| **雑魚モブ** | 自我を失った神の微小断片。本能だけで動く |
| **死霊術師** | 神の断片を再構成し、死者を使役する者 |
| **ボス** | 神の主要器官の守護者。かつての神の意志の残滓 |
| **最終ボス** | 神の心臓に宿る最後の意識——存在し続けたいという最後の衝動 |

### 2.5 神々（ゾティークの暗黒神殿）

| 神格 | 領域 | ゲーム内役割 |
|------|------|------------|
| **タサイドン** | 闇の七王国の主 | 呪いと取引のNPC |
| **モルディギアン** | 屍の神。全ての死者を所有 | 死後の行き先を決める存在 |
| **ヴェルガマ** | 運命の神 | ランダムイベントの裏の支配者 |

## 3. アートスタイルへの反映

### 確定方向: 油絵（A5/A7系統）

上記の世界観をアートに落とし込む際の指針：

| 要素 | ビジュアル指針 |
|------|-------------|
| **光源** | 暗い太陽。全ての光が黄昏色。複数光源（松明、魔法の残光、内発光） |
| **色調** | 琥珀、深紅、紫、暗黒。「神の血」の色が全体を支配 |
| **テクスチャ** | 全てが古い。ひび割れ、苔、錆、腐食。「力の弱体化」の物理的表現 |
| **構図** | 広大な崩壊。小さな人影と巨大な廃墟のスケール対比 |
| **筆致** | 美術館の油絵。焦点部分は精緻、周辺は程よく筆跡が残る |
| **陰影** | カラヴァッジョ的キアロスクーロ。光と影の境界は暗く深い |

## 4. 新ゲームモード: 深淵降下（シングルプレイ）

### コンセプト: 「神の心臓に到達せよ」

| 要素 | 設計 |
|------|------|
| **構造** | 10階層のダンジョン。各階層がボスで区切られる |
| **進行** | 不可逆。一度下ったら戻れない |
| **装備スロット** | 3つ: 武器 / 防具 / 遺物（アクセサリ） |
| **装備の入手** | ダンジョン内で発見。ショップは存在しない（文明がない） |
| **カードシステム** | 維持。ただし「死霊術」カテゴリ追加 |
| **エンディング** | 2つ: 救済（虚無への到達）/ 抗い（存在を続ける選択） |

### 装備スロット詳細

| スロット | 効果例 |
|---------|--------|
| **武器** | 攻撃力、射程、特殊効果（毒、炎、氷） |
| **防具** | HP上限、防御力、耐性 |
| **遺物** | パッシブ効果（HP自動回復、ノイズ軽減、視界拡大） |

装備は**神の断片**なので：
- 使い込むほど弱体化する（力の弱体化）
- 一部の装備は意志を持ち、装着者に影響を与える（呪い）
- 深層で見つかる装備ほど古く強力だが、呪いのリスクも高い
